# InstruqtでのLab
* [Instruqt](https://instruqt.com/)という仮想環境構築プラットフォームを用いてLabを行います。

* もし、Instuqtについてより詳細をしりたければ[tutorial](https://play.instruqt.com/instruqt/tracks/getting-started-with-instruqt)を参照ください。
